# 🛒 ShelfVision: Lenta Tech CV Pipeline

**ShelfVision** — это высокопроизводительный программный комплекс для автоматического мониторинга товарных полок и аудита ценников. Система спроектирована для обработки видеопотока с мобильных роботов-мерчандайзеров в условиях переменного освещения и активного движения.

---

## 🌟 Ключевые особенности

*   **Двухэтапная архитектура**: Синергия **YOLOv8** для сверхбыстрой детекции объектов и **PaddleOCR** для высокоточного извлечения текстовых данных.
*   **Оптимизация через ROI (Region of Interest)**: Динамическое выделение активной зоны кадра позволило сократить объем обрабатываемых данных и увеличить скорость работы в **~15 раз**.
*   **Глобальное маппирование (4K Scaling)**: Математический алгоритм пересчета координат из локального пространства ROI (1080p) в глобальную систему координат 4K (Ultra HD), соответствующую эталону разметки.
*   **Устойчивость к Motion Blur**: Применение фильтров повышения резкости (*Laplacian Sharpening*) для обеспечения корректного распознавания текста на кадрах, снятых в движении.

## 🏗 Архитектура решения

1.  **Motion Control**: Анализ видеопотока и пропуск статичных кадров для экономии вычислительных ресурсов.
2.  **Detection Phase**: YOLOv8 находит ценники и передает координаты в модуль трансформации.
3.  **OCR Phase**: Обрезка найденного ценника, наложение белых полей (Padding) и распознавание бренда, цены и скидок.
4.  **Logic Engine**: Парсинг сырого текста регулярными выражениями и автоматический расчет итоговой скидки.
5.  **Exporter**: Генерация CSV с кодировкой `utf-8-sig` для корректного отображения кириллицы в Excel.

## 🛠 Решенные инженерные задачи

### 1. Борьба с "Dependency Hell"
В ходе разработки была выявлена критическая ошибка сегментации (*Segmentation Fault*) в новых версиях PaddleOCR (3.x) при работе на Linux-системах без GPU. Проблема решена подбором стабильного «золотого стека» версий **PaddlePaddle 2.6.2 + PaddleOCR 2.8.1**, что гарантирует стабильность системы на CPU.

### 2. Точность координат в 4K (Ultra HD)
Поскольку видео обрабатывается в разрешении 1080p для обеспечения реального времени, а эталон разметки предоставлен в 4K, применен алгоритм обратной трансформации координат с учетом смещения ROI:

$$X_{global} = X_{local} \cdot Scale$$
$$Y_{global} = (Y_{local} + ROI_{offset}) \cdot Scale$$

### 3. Интеллектуальный парсинг цен
Разработан кастомный обработчик на базе регулярных выражений, который извлекает числовые значения цен до этапа очистки текста. Это позволило корректно обрабатывать цены с десятичными знаками и вычислять скидки даже при частичном зашумлении текстового поля.
